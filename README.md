## Chronos
Chronos was my first ever functioning AI Discord bot, which was based on Gemini 1.5 Flash. It was a dummy test bot that had a lot of features that were removed in later bots, like JSON memory and video recognition. A lot of seeminly 'good' features were removed due to big performance issues.

<img width="730" height="442" alt="image" src="https://github.com/user-attachments/assets/708ca30e-2082-495e-afa4-e489d08dbaf0" />
